# Prompt Skeleton

Context:
This is a proof-of-concept demo. It should show how one person plus AI can automate or simplify a real workflow for a small business or operations team.

Audience:
Non-technical small business owners / ops people. No enterprise sludge. No jargon. No fake SaaS grandstanding.

Task:
Build [specific demo/app/tool] based on [job spec/problem].

Constraints:

- Keep it simple and understandable.
- Use realistic sample data.
- No API keys or secrets committed.
- Include MIT license.
- Include README for non-developers.
- Credit OpenAI Codex and Claude.
- Treat this as a demo, not a production product.

Verification:
Don't finish until:

- The app runs locally.
- You have checked the UI in browser.
- There are no obvious console errors.
- README explains setup, purpose, and limits.
- .env is not committed.
- git status is clean or you explain what remains.

Output:
Summarise:

- What you built
- How to run it
- What files matter
- What you checked
- What I should inspect next
